# Project Documentation

Welcome to the Manager project documentation. This directory contains files related to the project's roadmap, change logs, and future plans.

## Directory Structure

- **`updates.json`**: A structured log of all major updates and fixes implemented in the application. Each step includes a description and the files affected.
- **`plans/`**: Contains markdown files outlining future feature expansions and design documents.
  - **`feature-expansion.md`**: The current roadmap for Trading, To-Dos, and Notification features.

## Latest Updates Summary

The application has recently implemented:
- **Global Currency Support**: Support for USD, EUR, GBP, INR, TZS, and more with real-time conversion.
- **Cloud Backup**: A robust syncing mechanism with visual feedback in the header and settings.
- **UX Improvements**: Reorganized Transaction Form and improved Pie Chart presentation for better clarity on mobile and desktop.
- **Integrated Calendar**: A new module that handles Events, Memories (with Spotify integration), Reminders, and Tasks.

## Contact & Maintenance

For any issues or feature requests, please refer to the `error.json` (tracked locally for quick fixes) or the `TODO.md` (if present) for internal tracking.
